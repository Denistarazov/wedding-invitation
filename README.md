# Свадебное приглашение

Статический сайт-приглашение для Дениса и Диляры.

## Как подключить почту

Форма уже настроена под FormSubmit. Чтобы ответы приходили на нужную почту,
сейчас форма настроена на:

```html
https://formsubmit.co/ajax/den484411@gmail.com
```

После первой отправки FormSubmit пришлет письмо для подтверждения адреса.
Нужно один раз подтвердить почту, и следующие ответы будут приходить в inbox.

## Локальный запуск

Можно открыть `index.html` напрямую в браузере или запустить локальный сервер:

```bash
python -m http.server 8000
```

После этого сайт будет доступен по адресу `http://localhost:8000`.
